# Automação de Testes Cypress - Sonia Wicki 🚀

Este projeto foi desenvolvido para o meu portfólio, demonstrando a automação de fluxos críticos utilizando **Cypress** e **JavaScript**.

## 🎯 Objetivo
Validar as funcionalidades de login e navegação do ambiente **ServeRest**, garantindo que as regras de negócio e mensagens de erro estejam funcionando corretamente.

## 🛠️ Tecnologias e Ferramentas
* **Framework:** Cypress
* **Linguagem:** JavaScript
* **IDE:** VS Code
* **Ambiente de Teste:** [ServeRest Front](https://front.serverest.dev/login)

## 🧪 Cenários de Teste (Implementados)
- [x] **Validação de Título:** Garante que a aplicação carregou corretamente.
- [x] **Login Negativo:** Verifica se o sistema exibe o alerta correto ao inserir credenciais inválidas.
